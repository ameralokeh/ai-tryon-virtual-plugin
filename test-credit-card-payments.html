üß™ Testing Steps

Complete Payment Flow Test

  1. Open the checkout modal: Go to Virtual Fitting Page
  2. Click "Try Virtual Fitting" or checkout button to open the React modal
  3. Verify payment methods: You should see "Credit Card (Test)" as an option
  4. Fill billing information: Enter valid customer details
  5. Select Credit Card payment method
  6. Enter test card details: Use 4242424242424242 for success
  7. Submit the form and verify successful payment processing
  8. Check WooCommerce orders: Verify order was created in admin
  9. Verify credits: Check that credits were added to customer account

Error Handling Test

  1. Test declined payment: Use card 4000000000000002
  2. Test validation: Try submitting with empty fields
  3. Test invalid card: Use a random card number
  4. Verify error messages: Check that appropriate errors are shown
  5. Test retry functionality: Verify users can retry after errors
